<h1>🎮 Sobre o projeto</h1>
Um jogo simples, cujo objetivo é adivinhar um determinado número aleatório na menor quantidade de tentativas. Trata-se de um projeto simples desenvolvido com HTML5, CSS e JavaScript básicos, desenvolvido com o intuito de por em prática conhecimentos em desenvolvimento web.

<h2>Get Started</h2>
<p>Clone o repositório em sua máquina e acesse o arquivo index.html com seu navegador.</p>
